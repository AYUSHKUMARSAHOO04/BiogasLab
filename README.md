# BiogasLab : Anaerobic Digestion Simulator

An interactive simulation platform for analyzing anaerobic digestion and biogas production using Monod kinetics and CSTR steady-state modeling.

## Features
- Dynamic biogas production simulation
- Methane (CH₄) yield prediction
- COD removal efficiency analysis
- pH and ammonia inhibition modeling
- Temperature effect analysis
- Hydraulic Retention Time (HRT) optimization
- Dynamic charts and process visualization

## Engineering Concepts Used
- Monod Growth Kinetics
- CSTR Steady-State Modeling
- Biomass Growth Kinetics
- Methane Yield Estimation
- COD Mass Balance
- Temperature Correction Factor
- Ammonia Inhibition Effects

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js

## Project Objective
To simulate anaerobic digestion performance and predict methane production under varying reactor and operating conditions for biogas process optimization.

## How to Run
1. Download the repository
2. Open `index.html`
3. Click **Run Simulation**
4. Adjust parameters and analyze outputs
